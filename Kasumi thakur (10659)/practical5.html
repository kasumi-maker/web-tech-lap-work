Learning HTML Basics

HTML (HyperText Markup Language) is the standard language used to create web pages. It helps us structure the content using headings, paragraphs, tables, and lists.

HTML pages are made up of elements that define different parts of the content. For example, the <h1> tag defines a heading, and the <p> tag defines a paragraph.

Using semantic elements such as <header>, <article>, and <footer> makes web pages easier to read and understand for both users and search engines.

Quick HTML Facts

  • HTML was first created in 1991.
  • It stands for HyperText Markup Language.
  • Every web page starts with <!DOCTYPE html>.
  • Tags are written inside angle brackets.
